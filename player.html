⚠ เปิดจาก file:// — CORS จะ block ทุก request
ต้องเปิดผ่าน HTTP server เท่านั้น:
• GitHub Pages — push ไฟล์นี้ขึ้น repo แล้วเปิด Pages
• VS Code Live Server — คลิกขวา → Open with Live Server
• Python local: python -m http.server 8080 แล้วเปิด http://localhost:8080
Cloudflare Worker URL
Worker จะถูกเรียกที่ {worker}/ หรือ {worker}/stream
หรือเปิดตรงๆ ด้วย ?w=https://your-worker.workers.dev
